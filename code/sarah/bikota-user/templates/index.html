BIKOTA

This project proposes an innovative bike renting service concept based on pseudo-anonymous, registration-free payment using Distributed Ledger Technology, namely IOTA. The only prerequisite is a Trinity Wallet.

Watch the video to see how the rental process works. If you are looking for available bikes, take a look at the map. And if you want to see the data collected during a rental session, just enter your session address.
